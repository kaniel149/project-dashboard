# Project Map - עיצוב מפת פרויקט אינטראקטיבית

> תאריך: 2026-02-03

## סקירה

מפה ויזואלית אינטראקטיבית שנפתחת מהדשבורד, מאפשרת להבין את המערכת, לנווט, לתכנן ולעבוד ישירות על פיצ'רים.

## שכבות המפה

| שכבה | צבע | מה מציגה |
|------|-----|----------|
| ארכיטקטורה | כחול | קומפוננטות, קבצים, תיקיות |
| מסכים | סגול | דפים, routes, user flows |
| פיצ'רים | ירוק | יכולות, מה המערכת עושה |
| TODOs | כתום | TODOs, באגים, שיפורים |
| סטטוס | אפור/ירוק | מה עובד, בעבודה, הושלם |

## פעולות (קליק ימני)

- פתח ב-VS Code
- פתח טרמינל
- הוסף משימה
- שנה סטטוס
- הוסף הערה

## סריקה אוטומטית

### מקורות מידע
- מבנה תיקיות (src/components, src/pages)
- קבצי React - קומפוננטות, imports
- Router - routes וניווטים
- הערות בקוד (TODO, FIXME, BUG)
- CLAUDE_STATE.md
- package.json

## ממשק

### סרגל עליון
- שם פרויקט
- Toggle לכל שכבה
- כפתור רענון
- כפתור שמירה

### קיצורי מקלדת
- Space + גרירה = הזזת קנבס
- Scroll = זום
- N = בלוק חדש
- T = TODO חדש
- E = הערה חדשה
- Cmd+Click = פתיחה ב-VS Code
- Cmd+S = שמירה

### צבעי סטטוס
- מתוכנן = אפור
- בעבודה = כחול + פעימה
- הושלם = ירוק
- באג = אדום
- תקוע = כתום
